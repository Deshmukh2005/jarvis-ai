# JARVIS-AI

A personal AI assistant inspired by Tony Stark’s JARVIS.  
It can perform voice-controlled tasks, text-to-speech, web automation, and features a glowing cinematic GUI.

## ✨ Features
- Voice recognition and command execution
- Text-to-speech responses
- Web automation (YouTube, Google, time queries)
- Futuristic Tkinter-based GUI with glowing effects

## ⚙️ Installation
```bash
git clone https://github.com/Deshmukh2005/jarvis-ai.git
cd jarvis-ai
pip install -r requirements.txt
